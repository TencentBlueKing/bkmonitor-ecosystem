# bkmonitor-ecosystem

## 📣 简介
<a href="https://github.com/TencentBlueKing/bkmonitor-ecosystem.git" target="_blank">bkmonitor-ecosystem</a> 提供「最小化」开箱即用的观测数据接入 demo，帮助用户快速入门蓝鲸观测平台功能。

## 📦 开箱即用
